[ FR ]
Quelles nouvelles :
Cette version corrige une difficulté de déploiement de SecretManager. Désormais, vous un fichier spécifique pour localiser les ressources de SecretManager. Par défaut, les ressources sont correctement localisées.

Première installation :
Déziper "SecretManager_v0.5-0.zip" dans le "DocumentRoot" de votre Serveur Apache.
Ensuite, lire le "Guide d'installation" dans le répertoire "Documentations" dans l'arborescence de SecretManager.

Mise à jour de v0.4-0 à v0.5-0:
Déziper "upd_SecretManager_v0.5-0.zip" dans le précédent répertoire d'installation de votre "SecretManager".



[ EN ]
What's news :
This version fixe a SecretManager deployment difficulty. Now, you've got a specific file to locate SecretManager ressources.
By default, the ressources are in good location.

First Installation :
Unzip "SecretManager_v0.5-0.zip" in your Apache Server "DocumentRoot".
Read the "intallation guide" in "Documentation" directory

Upgrade v0.4-0 to v0.5-0:
Unzip "SecretManager_upd_v0.5-0.zip" in your "SecretManager" directory (precedent install).
