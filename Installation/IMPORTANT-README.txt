First Installation :
Unzip "SecretManager_v0.2-0.zip" in your Apache Server "DocumentRoot".
Read the "intallation guide" in "Documentation" directory

Upgrade v0.1-7 to v0.2-0:
Unzip "SecretManager_upd_v0.2-0.zip" in your "SecretManager" directory (precedent install).
In the "installation guide" just follow "SecretServer" chapters.
